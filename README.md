# gt_surveillance

download bam files from TCGA.

## Dependency

python >= 2.7

Python packages

 - drmaa

External script or binary.

 - gtdownload
 - xmlsplitter.pl

## install

```
git clone https://github.com/aokad/gt_surveillance.git
```

## Run

```
python {install path}/gt_surveillance.py {output root path} {key file} {manifest download from TCGA}
```
